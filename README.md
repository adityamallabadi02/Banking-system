# Banking-system
This project is a console-based Bank Management System developed using the C programming language. It demonstrates the use of structures, binary file handling, and CRUD operations to manage bank account records efficiently.

The program allows users to create accounts, view account details, search accounts, deposit and withdraw money, and delete account records. All data is stored permanently using binary files, ensuring fast and secure storage.
